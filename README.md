# Conde Builds

Static portfolio website for Marcos M. Conde.

## Local Preview

Open `index.html` in a browser, or serve the folder with any static server.

## Netlify

This site has no build step. In Netlify:

- Base directory: `conde-builds-site`
- Build command: leave blank
- Publish directory: `.`

After deployment, connect `conde-builds.com` in Netlify domain settings and follow the DNS records Netlify provides.
